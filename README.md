# qiankun-vue-ts-demo
一个基于乾坤（qiankun）+vue+typescript的微前端demo

## 应用启动

#### 主应用启动

```javascript
cd  main

npm install

npm run serve
```
#### 子应用1启动

```javascript
cd  subone

npm install

npm run serve
```

#### 子应用2启动

```javascript
cd  subtwo

npm install

npm run serve
```

#### 浏览器打开链接
```javascript
http://localhost:8080/
```
